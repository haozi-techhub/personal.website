# 03 · 首页全屏滚动蓝图

> 技术方案：CSS `scroll-snap-type: y mandatory` + section `100vh`，配合 `IntersectionObserver` 同步导航高亮。
> 不引入 fullPage.js 等库，保持轻量。

**要点**
- 左文右图，黄金比例（约 5:7）
- 「Hello!」用 Fraunces 斜体，加 `--acid` 高亮块（参考图 5）
- 「任文浩 / Haozi」中英大字混排，中文占主，英文小一号
- 自我陈述精简到 2 行（建议文案见 07 文档）
- CTA 两个：主按钮跳摄影，次按钮跳简历
- 右上角小贴纸：星形「2025」徽章
- 底部「↓ scroll · 1/5」提示，是 snap 进度的暗示

## 屏 2 · `#photos` — 摄影预览

```
┌─────────────────────────────────────────────────────┐
│  📷 PHOTOGRAPHS / 摄影                01           │
│  See the world through my lens.                     │
│                                                     │
│  ┌───────┐  ┌──────────┐  ┌───────┐  ┌──────┐     │
│  │       │  │          │  │       │  │ +    │     │
│  │ 精选 1│  │ 精选 2   │  │ 精选 3│  │ 占位 │     │

│  └───────┘  └──────────┘  └───────┘  └──────┘     │
│       (错落 mosaic, 不规则高度)                     │

│       ➜ 进入摄影完整页 (大箭头按钮)                  │
```

**要点**
- 标题左侧大写编号「01」（编辑感）
- 3 张精选图 + 1 个「+」占位卡（**冗余位**）
- 卡片高度交错（mosaic），不要等高
- 颜色暗示：`--blush` 作为本屏强调色（小色块或描边）
